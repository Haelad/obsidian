# Redirect Функция
### следует знать что:
 - 
	возвращает HTTP-ответ для перенаправления пользователя на другой URL
 - 
	Функции можно создать дополнительный параметр `permanent`, который отвечает за код состояния ответа HTTP

## [Коды состояния ответа HTTP](https://developer.mozilla.org/ru/docs/Web/HTTP/Status) (Параметр Redirect) 
###### Основные:
- 301 - страница перемещена на другой постоянный URL адрес
- 302 - страница перемещена временно на другой URL адрес

#### Синтаксис:

`redirect('url', permanent=True)`

#### Параметры:

- `url` - маршрутизатор
- `permanent=` - параметр функции, который влияет на код ответа HTTP

# Reverse Функция
<h3>следует знать что:</h3>
**1. 
	 Reverse строит динамический url маршрут по имени маршрутизатора

**2. 
	Reverse можно указать дополнительные аргументы


`reverse('view_name', args=(''))`

#### Параметры:

- `view_name` -  имя маршрутизатора
- `args=('') `- аргументы, которые являются коллекцией параметров маршрутизатора
