Кортежи (tuples) — это неизменяемые упорядоченные коллекции объектов. Они похожи на списки, но в отличие от них, кортежи не могут быть изменены после создания. Это делает их идеальными для хранения данных, которые не должны изменяться, например, конфигурационные данные или координаты точек.
Кортежи часто используются в Python для группировки данных, возвращения нескольких значений из функций и в операциях, где важна неизменяемость данных.

Вот некоторые основные операции и методы, связанные с кортежами в Python:

- Создание кортежа: `my_tuple = (1, 2, 3)` или `my_tuple = 1, 2, 3`
- Доступ к элементам: `my_tuple[0]` возвращает первый элемент кортежа
- Срезы: `my_tuple[1:]` возвращает кортеж, содержащий все элементы, начиная со второго
- Конкатенация: `my_tuple + another_tuple` объединяет два кортежа
- Умножение: `my_tuple * 3` создает новый кортеж, повторяя его элементы
- Методы:
    - `count(x)`: **возвращает количество раз, которое `x` встречается в кортеже**
    - `index(x)`: **возвращает индекс первого вхождения `x` в кортеже**

###пример с кодом:
```python
# Создание кортежа
coordinates = (10.0, 20.0)

# Доступ к элементам
x = coordinates[0]  # x будет равен 10.0
y = coordinates[1]  # y будет равен 20.0

# Конкатенация кортежей
more_coordinates = (30.0, 40.0)
all_coordinates = coordinates + more_coordinates  # (10.0, 20.0, 30.0, 40.0)

# Умножение кортежа
doubled_coordinates = coordinates * 2  # (10.0, 20.0, 10.0, 20.0)

# Использование методов
count = coordinates.count(10.0)  # Выведет: 1
index = coordinates.index(20.0)  # Выведет: 1
```
