#### Список в Python – это структура данных, которая может хранить последовательность элементов. Эти элементы могут быть любого типа, например, чисел, строк или даже других листов. Списки в Python очень гибкие и удобные для использования, потому что:

###### - Вы можете добавлять и удалять элементы.
###### - Можно обращаться к элементам по индексу.
###### - Списки можно изменять после создания.

Пример списка в Python:

```python
list = [1, 2, 3, 'Привет', 4.5]
```
> В этом примере `list` содержит целые числа, строку и число с плавающей точкой.

## Методы

Методы списка в Python — это встроенные функции, которые предоставляются для работы со списками, одной из основных структур данных в этом языке программирования. Списки в Python могут содержать элементы различных типов и предоставляют множество методов для управления этими элементами.

Вот некоторые из наиболее часто используемых методов списка:

- `append(x)`: **добавляет элемент `x` в конец списка**.
- `extend(iterable)`: **расширяет список**, добавляя в конец все элементы из `iterable`.
- `insert(i, x)`: **вставляет элемент `x` на позицию `i`**, сдвигая остальные элементы вправо.
- `remove(x)`: **удаляет первый элемент `x` из списка**.
- `pop([i])`: **удаляет элемент по индексу `i`** (по умолчанию последний) и возвращает его.
- `clear()`: **очищает список**, удаляя все его элементы.
- `index(x[, start[, end]])`: **возвращает индекс первого элемента `x`** (начиная с `start` и заканчивая `end`).
- `count(x)`: **возвращает количество элементов `x` в списке**.
- `sort(key=None, reverse=False)`: **сортирует элементы списка** (опционально с ключом `key` и в обратном порядке, если `reverse=True`).
- `reverse()`: **меняет элементы списка на обратный порядок**.
- `copy()`: **возвращает копию списка**.

### Метод append()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
people.append("Masha")  
print(people)
# выведет:
# ['Victor', 'Misha', 'Pasha', 'Masha']
```
>**Метод append() добавляет в конец списка новый(е) элемент(ы).**

### Метод clear()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
people.clear()  
print(people)
# выведет:
# []
```
> **Метод clear() очистит все элементы списка, оставив его пустым.**

### Метод remove()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
people.remove("Misha")  
print(people)
# введет:
# ['Victor', 'Pasha']
```
> **Метод remove() уберет определенный, указанный элемент из текущего списка.**

### Метод copy()
```python
people: list[str] = ["Victor", ["Misha", "Pasha"], 'Elon']  
copied_people: list[str] = people.copy()  
copied_people.remove("Victor")  
# copied_people[1][1] = "Dog"  
print(copied_people)
# введет:
# [['Misha', 'Pasha'], 'Elon']
```
> **Метод copy() создаст точную копию списка с другим адресом в памяти, которая не влияет на основную, что делает его совершенно другим объектом.**

### Метод pop()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
popped_Victor = people.pop(0)  
print(popped_Victor)
# введет:
# Victor
```
> **Метод pop() вынимает из списка элемент по индексу, если индекс не установлен** *"удалит"* **последний элемент списка.**

### Метод reverse()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
people.reverse()  
print(people)
```
>**возвращает тот же список с изменёнными индексами элементов** *(расположенными в обратном порядке).*
   **Индексы элементов меняются таким образом, что первый элемент становится последним, второй - предпоследним, и так далее.**

### Метод insert()

```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
people.insert(0, "Vanya")  
print(people)
# Выведит:
# ['Victor', 'Misha', 'Pasha', 'Vanya']
```
> **Метод insert() добавляет элемент по индексу, если на месте индекса уже стоит элемент, он подвинет его на следующий по счету в право.** 

### Метод index()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
print(people.index('Victor'))
# Выведит:
# 0
```
> **Метод index() возвращает индекс по элементу**

### Метод extend()
```python
people: list[str] = ["Victor", "Misha", "Pasha"]  
new_people: list[str] = ["Maria", 'Sarah', 'Joseph']  
people.extend([new_people])  
print(people)
# Выведит:
# ['Victor', 'Misha', 'Pasha', ['Maria', 'Sarah', 'Joseph']]
```
> **расширяет, добавляет итерируемый объект в список**

### Метод count()
```python
people: list[str] = ["Victor", "Misha", "Pasha", 'Victor']  
Victors = people.count('Victor')  
print(Victors)
```
> **считает количество объектов в списке**

### Метод sort()

###### Метод `sort()` в Python сортирует список строк **по алфавиту** и в соответствии с лексикографическим порядком. Это означает, что строки сравниваются посимвольно, начиная с первого символа, и сортируются в порядке возрастания их кодов ASCII или Unicode. Строки с одинаковыми начальными символами будут сортироваться далее по второму символу, и так далее.

###### Если в списке есть строки с разным регистром, то по умолчанию заглавные буквы (большие буквы) будут расположены перед строчными (маленькими буквами), так как их коды в ASCII меньше.
###### Также метод `sort()` можно настроить с помощью параметра `key`, который позволяет указать функцию для определения порядка сортировки

Вот пример сортировки списка строк:

```python
words = ['banana', 'apple', 'cherry'] 
words.sort() 
print(words) 
# Выведет: 
# ['apple', 'banana', 'cherry']
```
>и такой же, но с заглавными буквами в некоторых словах
```python
words = ['Banana', 'apple', 'Cherry'] 
words.sort() 
print(words) 
# Выведет: 
# ['Banana', 'Cherry', 'apple']`
```


Чтобы сортировать строки вне зависимости от регистра, можно использовать параметр `key` с функцией `str.lower`:
```python
words = ['Banana', 'apple', 'Cherry'] 
words.sort(key=str.lower)
print(words)
# Выведет:
# ['apple', 'Banana', 'Cherry']
```

а вот как он сортирует числа
```python
numbers = [3, 1, 4, 1, 5, 9, 2]
numbers.sort()
print(numbers)  # Выведет: [1, 1, 2, 3, 4, 5, 9]
```
он сортирует их по возрастанию

Если вы хотите отсортировать список в обратном порядке, используйте `sort(reverse=True)`:
```python
numbers.sort(reverse=True)
print(numbers)  # Выведет: [9, 5, 4, 3, 2, 1, 1]
```
