# Структуры данных в Python 

это **способы организации и хранения данных в памяти компьютера**. В Python существует множество встроенных структур данных, таких как списки, кортежи, словари и множества, которые предоставляют различные способы организации и доступа к данным.


### пример Структуры данных:

```python
list = [3,412,'hi',True]
tuple = (3,412,'hi',True)
dict = {'name': 'value'}
set = {42, 42, 2, 4, 624, True, True, 'this is set'}
```
