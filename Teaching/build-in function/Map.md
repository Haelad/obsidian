# Функция Map()
### следует знать что:

**1. 
	map возвращает итератор, в котором к каждому итерируемому объекту применяется функция.** 
2.
	 Функция map, принимает на вход:
	 Функцию, которой передают каждый элемент массива и сам Массив, к которому будет применятся функция
**3. 
	функция map используется для упрощения излишнего, шаблонного кода**
	
**4. 
	К недостаткам `map()` можно отнести «ленивую оценку» (Lazy Evaluation). 
	нам возвращается ссылка на итератор.
	Пока мы не обернем ее в `list()` или другой метод, обработка функцией-дочкой еще не произошла.

#### Синтаксис:

**Map Function** - `map(function, iterable)`

#### Параметры:
- `function` - Функция
- `iterable` - коллекция, которую можно

## применение самодельной функции:
```python
def make_even(num):
	if num % 2 == 1:
		return num + 1
	else:
		return num

x = [412, 564, 432, 321, 663, 513, 746, 342, 111, 347, 429, 743]

y = list(map(make_even, x))

print(y)



```

## self() для обращения объекта к себе:
```python
class Foo():
    
    def __init__(self):
        self.factor = 2
        
    def multiply(self, num):
        return num * self.factor

    def show(self, ls):
        print(list(map(self.multiply, ls)))

>>> f = Foo()
>>> f.show([1,2,3])
... [2, 4, 6]

```
> В данном случае `self` позволяет перемножить элементы списка на 2


## map и анонимная функция:
```python
ls = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(list(map(lambda x: x * x, ls)))

# [1, 4, 9, 16, 25, 36, 49, 64, 81]

```
> анонимная функция или lambda функция используется для укрощение и упрощение написания «Шаблонного кода» (**Boilerplate code**)
 


## Строки:
```python
import string

my_string = 'TProger'
list(map(string.capwords, my_string)
# ['T', 'P', 'R', 'O', 'G', 'E', 'R']



```
>`map()` использует «дочернюю» функцию на любом объекте, где применим цикл `for`. Мы можем применять ее к любому итерируемому объекту, даже к strings.
>затем мы импортируем модуль `string`, чтобы использовать метод `capwords()` — приведение к верхнему регистру
