# Функция filter

**принимает в качестве аргументов функцию и объект, элементы которого следует отфильтровать**

***следует знать что:***
- Если фильтрующая функция `func` вернёт `False`, то элемент последовательности `iterable` не попадёт в результат выполнения функции `filter()`.

- Если фильтрующая функция `func` вернет `None`, то считается что требуется применить тождественное действие `(item for item in iterable if item)`, таким образом все элементы, оцениваемые как `False` будут отфильтрованы.

- **для правильного использования функции filter() нужно входе вычислений/сравнений над каждым элементом** передаваемой последовательности получать `True` или 1 для элементов которые нужно отобрать и `False` или 0 для элементов, которые нужно отбросить.

#### Синтаксис:

`filter(func, iterable)`

#### Параметры:

- `func `- [функция](https://docs-python.ru/tutorial/opredelenie-funktsij-python/ "Определение функции в Python."), которая принимает элемент фильтруемого объекта и должна вернуть [`bool`](https://docs-python.ru/tutorial/osnovnye-vstroennye-tipy-python/bool-logicheskij-tip-dannyh/ "Логический тип данных bool в Python.") значение *(True or False)*
- ` iterable `- [последовательность](https://docs-python.ru/tutorial/osnovnye-vstroennye-tipy-python/tipy-posledovatelnostej/ "Типы последовательностей в Python.") или объект поддерживающий [итерирование](https://docs-python.ru/tutorial/osnovnye-vstroennye-tipy-python/tip-dannyh-iterator-iterator/ "Итератор Iterator, протокол итератора в Python.") *(простыми словами - коллекция).*

### пример с кодом:
```python
num = list(range(0, 27))

# определяем отдельную функцию которая
# отбирает четные элементы > 18.
def ff(x):
  if x > 18 and not x % 2:
    return True
  else:
    return False

f = list(filter(ff, num))
print(f)
# [20, 22, 24, 26]
```


### Применение лямбда-функции в качестве фильтра последовательности.

```python
# имеем числовую последовательность
num = [1, 2.0, 3.1, 4, 5, 6, 7.9]

# использование операции сравнения
# отбираем числа > 4
>>> f = filter(lambda x: x > 4, num)
>>> list(f)
# [5, 6, 7.9]

# использование оператора идентичности `is`
# отбираем только целые числа
>>> f = filter(lambda x: type(x) is int, num)
>>> list(f)
# [1, 4, 5, 6]
```

### Удаление ложных значений (`False`, `None`,  `0 `, `''`) из списка с помощью `filter()`.

```python
lst = [0, 1, False, 2, '', 3, 'a', 's', 34]
a = list(filter(None, lst))
print(a)
# [1, 2, 3, 'a', 's', 34]
```
