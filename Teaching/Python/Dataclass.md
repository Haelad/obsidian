# Dataclass 
## это декоратор класса, использующейся для репрезентации объектов класса. 
## используются для хранения данных без необходимости писать много *шаблонного кода*.

### к примеру:
```python
from dataclasses import dataclass

@dataclass
class Fruit:
	fruit: str
	calories: int | float

banana = Fruit("banana", 12)
print(banana)
# выведет: Fruit(fruit='banana', callories=12)
```
*примечание* - если бы мы не использовали dataclass то мы бы вывели адрес класса
### В нем автоматически имеются специальные методы, такие как `__init__()` и `__repr__()`, которые обычно приходится писать вручную.
### Это делает код более чистым и легким для понимания, поскольку уменьшает количество повторяющегося кода.
# Keywords

## В dataclass есть много кейвордов, которые подходят для различных сценариев

- ## init
	Если установлено в `True` (по умолчанию), создается метод `__init__()`, который автоматически инициализирует атрибуты класса.
- ## repr 
	Если установлено в `True` (по умолчанию), создается метод `__repr__()`, который возвращает строку репрезентации объекта класса.
- ## eq
	Если установлено в `True` (по умолчанию), создается метод `__eq__()`, который определяет поведение оператора равенства `==`.
- ## order
	Если установлено в `True`, создаются методы `__lt__()`, `__le__()`, `__gt__()`, и `__ge__()`, которые позволяют сравнивать объекты класса.
- ## unsafe_hash
	Если установлено в `True`, создается метод `__hash__()`. Это следует использовать с осторожностью, так как неправильное использование может нарушить свойство hashable объекта.
- ## frozen
	Если установлено в `True`, создается неизменяемый (_immutable_) объект. Попытка изменить атрибуты объекта приведет к возникновению исключения `FrozenInstanceError`.
- ## match_args
	Если установлено в `True` (по умолчанию в Python 3.10 и выше), это позволяет использовать объекты `dataclass` с функцией `match/case` в Python.
- ## kw_only
	Если установлено в `True`, все поля, определенные после этого параметра, должны быть инициализированы только как ключевые слова.
	
- дополнительно
	- **field**: Функция `field()` используется для определения и настройки атрибутов класса данных. С помощью этой функции можно задать параметры, такие как `default`, `default_factory`, `init`, `repr`, `compare`, `hash`, и `metadata`.
	
		- **default**: Параметр `default` в функции `field()` используется для установки значения по умолчанию для поля.
		
		- **default_factory**: Параметр `default_factory` в функции `field()` используется для указания функции, которая будет вызываться для создания значения по умолчанию, если значение не было предоставлено при инициализации.
	
		- **metadata**: Параметр `metadata` в функции `field()` позволяет добавить произвольные метаданные к полю.
