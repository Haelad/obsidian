
Docker - это инструмент, что бы запаковать свое приложение в некий контейнер.

Docker строится на 3 столбах:

1. [Dockerfile](Dockerfile.md)
	 чертеж для образа нашего приложения (того как оно будет выглядеть и что в нем будет)

2. [Image](Image.md) 
	 шаблон (или образ) для запуска контейнера, именно отсюда будет браться информация о образе контейнера и отсюда же будет копироваться 

3. [Container](Container.md) 
	работающий процесс нашего приложения
